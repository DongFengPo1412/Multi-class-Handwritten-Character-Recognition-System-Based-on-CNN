# 基于 CNN 的多类别手写字符识别与修正系统
(Multi-class Handwritten Character Recognition & Correction System Based on CNN)

[简体中文](README.md) | [English](README_EN.md) | [日本語](README_JA.md)

---

## 🌟 项目简介

本系统是针对大三《智能控制》课程设计要求（手写字符识别任务）进行深度开发的高性能手写体 OCR 识别与修正系统。系统在**保持底层自定义 CNN 卷积神经网络架构不变**的前提下，借鉴了现代工业级 OCR（如 QQ 文字提取）的成熟算法思想，重点在**“图像预处理”**与**“识别结果修正”**两个环节进行了算法升级：
* **核心识别**：保留自定义的 3 层卷积块 `HandwrittenCNN` 模型，并应用 TTA（测试时增强）多采样机制。
* **抗干扰预处理**：首创**背景光差分补偿去阴影算法**与**自适应边界框聚类合并算法**。
* **智能结果修正**：首创**联合概率词典解码器 (Joint Probability Lexicon Decoder)** 与**空间几何校验（高宽比/相对高度）**。
* **硬件与交互**：支持电脑自带摄像头与外设的**一键热切换 (`c` 键)**，并对置信度低的模糊结果提供 **Top-3 备选结果输出**。

## 🛠️ 核心算法设计（完美对应课设要求）

### 1. 采集与图像预处理（基本要求1）
在真实摄像头演示时，环境阴影（如手机、手部投影）会导致二值化产生大量黑斑。
* **背景光消除算法**：利用大核高斯滤波器（$51 \times 51$）对 ROI 区域进行模糊，反向估算出背景的“光照强度分布图”（Illumination Map）。然后通过 `cv2.divide` 将原图除以背景光强分布图，得到完美的白底黑字笔画，彻底抹除阴影。
* **CLAHE & 自适应阈值**：对去光照图像使用限制对比度自适应直方图均衡化（CLAHE）以及自适应阈值化（Adaptive Thresholding），提取出平滑、无噪点的笔画二值图。

### 2. 特征选择与分割识别（基本要求2）
* **自适应边界框聚类合并 (Box Merging)**：
  - 自动检测并聚类相邻的轮廓边界框。
  - **垂直合并**：若两框在 X 轴上重叠度大且垂直距离小于设定的阈值，则融合成一个框（完美解决小写字母 `i`、`j` 字符的点与身体被割裂的问题）。
  - **水平邻近合并**：自动融合由于签字笔书写过快或过细导致的笔画断裂字符（如 `A` 字符中未连上的笔画）。
* **EMNIST 对齐特征提取**：计算融合框的物理质心（Image Moments），将其平移对齐到 $28 \times 28$ 规范画布的中心（$14, 14$），然后送入 CNN 网络识别。

### 3. 识别结果修正算法（基本要求3 & 核心优化点）
CNN 对手写容易混淆的 `oO0`、`1Il`、`2Zz`、`5Ss` 等字符，在 $28 \times 28$ 归一化后纯靠字形极难区分。本系统引入**联合概率词典解码**与**几何校验**进行全局校正：
* **上下文类别划分**：自动判断字符序列是“英文单词”还是“数字/混合编码”。
* **联合概率词典解码 (MAP)**：在英文单词上下文下，不使用简单的 ArgMax（最大概率）直接硬识别，而是将所有位置上的 62 维 Softmax 概率矩阵与英文词典（10,000个常用词）中长度匹配的单词进行联合概率对数打分：
  $$\text{Score}(W) = \sum_{i=1}^{N} \ln ( P(c_i.lower() | x_i) + P(c_i.upper() | x_i) )$$
  选取概率积最大的词作为修正输出（例如将 CNN 原始输出 `he11O` 修正为单词 `hello`）。
* **高宽比过滤 `0` 与 `O`**：通过长宽比 `w/h < 0.52` 对 `0` 进行置信度加权，反之倾向于字母 `O/o`。
* **相对高度比纠正大小写**：计算字符高度与行内最高字符的比例 $r = h_i / \max(H)$。若 $r < 0.78$ 且判定为对称字（如 `Z`），则纠正为小写 `z`，反之保留大写。

## 🚀 运行与操作说明

### 1. 启动实时摄像头提取系统
```bash
python camera_detect.py
```
* **动态热切换摄像头 (`c` 键)**：
  - 启动后系统会自动扫描并打开第 0 个摄像头（通常是内置摄像头）。
  - 若系统连接了多个摄像头（如 USB Webcam 与 Built-in Camera），只需随时在控制台或画面聚焦时按下 **`c` 键**，系统会自动在所有可用索引之间循环切换，无需重启程序。
* **框选与顺序识别 (空格键)**：
  - 将写有数字或单词的白纸置于红色检测框内（建议使用 0.5mm 黑色签字笔）。
  - 在 `AI Vision` 窗口中能看到去阴影后的完美笔画。
  - 按下 **【空格键】**，控制台和主界面会瞬间输出 `CNN 原始识别` 与 `算法修正后最终结果`。
* **不确定候选显示**：
  - 若有字符的置信度较低（概率小于 80% 或前两名差距小于 20%），在命令行和窗口下方会打印黄色的 `⚠️ 模糊字符警示`（包含前 3 个最可能的候选字及概率，如：`Char #3: 'l' (55.0%), '1' (35.0%), 'I' (5.0%)`）。
* **退出 (`q` 键)**：
  - 按下 **`q`** 键，释放视频流并安全关闭窗口。

### 2. 启动可视化离线测试
```bash
python predict.py
```
- 会在控制台打印模拟的 `hello`, `zoom`, `class`, `2026`, `10850` 经过纠错前后的比对报告，展示概率词典与几何后处理的强大效果。
- 随后会依次从 EMNIST 测试集中随机载入成组图片，弹出可视化对比界面（绿色代表正确，红色代表错误，橙色代表存在歧义）。按回车键切换下一组。
